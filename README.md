# Simple Radio Menu (FiveM)

Et simpelt radiomenu script til FiveM.
Lavet det fordi jeg kedede mig, og manglede noget clean uden alt muligt unødvendigt.

## Features

* Simpelt UI
* Bruger **pma-voice** til radio animationer

## Dependency

* pma-voice

## Kendte problemer

* Hvis animationer ikke virker → tjek pma-voice er korrekt sat op
* Radio kan opføre sig mærkeligt hvis andre voice scripts konflikter
* Ikke testet med alle frameworks
